About Our Project

Our IoT Cyber Attacks Detection project aims to safeguard connected devices by identifying and mitigating cyber threats in real-time. Leveraging advanced machine learning algorithms, we monitor network traffic to detect anomalies and potential attacks that could compromise IoT ecosystems.

We focus on key network attributes like Src_Port, Dst_Port, Flow_Duration, and others to create a robust security solution that is both efficient and scalable for various IoT environments.
